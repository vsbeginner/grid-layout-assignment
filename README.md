# grid-layout-assignment
Nature Theme Grid Layout
Project Overview

1) A responsive webpage showcasing the beauty of nature.

2) Built using HTML5 and CSS Grid.

3) Designed to adapt seamlessly to desktop, tablet, and mobile devices.

4) Features hero section, content area, banner, image section, and extra highlights.

Key Features

1) Fully responsive layout using CSS Grid and media queries.

2) Nature-themed design with forest, mountains, and river backgrounds.

3) Smooth hover effects for menu links and buttons.

4) Animated hero text for engaging user experience.

5) Separate sections for structured content and visual appeal.

Technologies Used
1) HTML5 – For semantic structure.
   
2) CSS3 – For styling, grid layout, responsiveness, and animations.
   
3) Media Queries – For tablet and mobile responsiveness.
